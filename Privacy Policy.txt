PRIVACY POLICY
Discord Bot System (Economy Bot, Backup Bot, Verification Bot)

Last Updated: March 2026

This Privacy Policy explains how the Discord bots operated by the Bot Owner (collectively referred to as the "Bots") collect, store, and use information when users interact with the Bots within Discord servers.

By using the Bots, you consent to the data practices described in this Privacy Policy.
If you do not agree with this Privacy Policy, you must discontinue use of the Bots.

1. Bots Covered by this Policy

This Privacy Policy applies to the following Discord bots:

Economy Bot
A role-playing economy system that provides features such as virtual currency, inventories, gambling games, leveling systems, combat mechanics, social features, and user profiles.

Server Backup Bot
A server management utility designed to create backups of Discord server configurations including roles, channels, permissions, server settings, and message data.

Verification & Security Bot
A verification and moderation system used to verify users, log moderation events, detect suspicious activity, and help administrators manage server security.

2. Information We Collect

The Bots may collect and store certain information required for their functionality.

User Information
• Discord User ID
• Discord Username and Tag
• Server Nickname (if applicable)
• Economy balances and gameplay statistics
• Inventory items and user progression data
• User profile information set through bot commands

Server Information
• Server ID
• Channel IDs
• Role IDs
• Server configuration settings used by the bots

Verification Information (Verification Bot)
• IP address
• approximate geographic location (city/country)
• verification answers provided by the user
• verification timestamps
• device or browser user-agent information
• risk detection scores used to prevent abuse

Backup Information (Backup Bot)
• server structure information
• channel configurations
• role configurations
• permission settings
• emojis and webhooks
• limited message history depending on backup type

3. How Information Is Used

Collected data may be used for:
• operating bot features and commands
• maintaining the economy system and player progression
• enabling RPG gameplay features
• verifying users and preventing malicious activity
• creating server backups and restoring configurations
• generating moderation and activity logs
• improving reliability and performance of the Bots

4. Data Storage

Data collected by the Bots may be stored in secure databases such as:
• MongoDB databases
• SQLite databases
• local or cloud-based storage systems

Backup files may be stored temporarily or permanently depending on server configuration and retention policies.

5. Data Retention

Data may be retained for as long as it is required to operate the Bots.

Retention periods may vary depending on the type of data:

Economy Data
Stored to maintain player progression and statistics.

Verification Data
Stored for moderation and security analysis.

Backup Data
Stored until manually deleted or automatically removed according to backup retention policies.

6. Data Sharing

The Bot Owner does not sell, trade, or rent user data to third parties.

Data may only be shared in the following situations:
• when required by law
• when necessary to prevent abuse or illegal activity
• when required for technical services that host bot infrastructure

7. Security

Reasonable measures are taken to protect stored data from unauthorized access, including:
• restricted database access
• secure server environments
• monitoring for suspicious activity

However, no online system can guarantee complete security.

8. User Rights

Users may request information about data stored by the Bots.

Where possible, users or server administrators may request:
• deletion of stored user data
• removal of server data
• removal of backups created by the Backup Bot

Requests must be made through the Bot Owner or server administrators.

9. Third-Party Services

The Bots rely on third-party services including:
• Discord API
• hosting providers
• database services

These services operate under their own privacy policies.

10. Changes to this Privacy Policy

The Bot Owner may update this Privacy Policy at any time to reflect improvements or changes to the Bots.

Users are encouraged to review this policy periodically. Continued use of the Bots after changes indicates acceptance of the updated policy.

11. Contact

If you have questions regarding this Privacy Policy or the handling of your data, please contact the Bot Owner or the administrators of the Discord server where the Bots are used.

By using the Bots, you acknowledge that you have read and understood this Privacy Policy and agree to its terms.
